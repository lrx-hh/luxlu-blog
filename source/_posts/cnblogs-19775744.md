<h2 id="1-安装-windows-terminal">1. 安装 Windows Terminal</h2>
<h3 id="方法一microsoft-store推荐">方法一：Microsoft Store（推荐）</h3>
<p>在 Microsoft Store 搜索 <strong>Windows Terminal</strong> 安装即可。</p>
<h3 id="方法二命令行安装">方法二：命令行安装</h3>
<pre><code class="language-powershell">winget install Microsoft.WindowsTerminal
</code></pre>
<hr>
<h2 id="2-安装-powershell-7">2. 安装 PowerShell 7</h2>
<p>Windows 自带的是 PowerShell 5，建议升级到最新的 PowerShell 7。</p>
<pre><code class="language-powershell">winget install Microsoft.PowerShell
</code></pre>
<p>安装完成后，在 Windows Terminal 中将 PowerShell 7 设为默认终端：</p>
<ol>
<li>打开 Windows Terminal</li>
<li>按 <code>Ctrl + ,</code> 打开设置</li>
<li>左侧点击 <strong>启动</strong></li>
<li>默认配置文件改为 <strong>PowerShell</strong>（不是 Windows PowerShell）</li>
</ol>

<h2 id="6-安装常用-powershell-模块">6. 安装常用 PowerShell 模块</h2>
<h3 id="61-psreadline命令行增强">6.1 PSReadLine（命令行增强）</h3>
<p>提供语法高亮、历史搜索、智能补全等功能。</p>
<pre><code class="language-powershell">Install-Module PSReadLine -Force -SkipPublisherCheck
</code></pre>
<p>在 Profile 中添加配置：</p>
<pre><code class="language-powershell"># 语法高亮
Set-PSReadLineOption -PredictionSource History
Set-PSReadLineOption -PredictionViewStyle ListView

# 快捷键
Set-PSReadLineKeyHandler -Key Tab -Function MenuComplete
Set-PSReadLineKeyHandler -Key UpArrow -Function HistorySearchBackward
Set-PSReadLineKeyHandler -Key DownArrow -Function HistorySearchForward
</code></pre>
<blockquote>
<p>⚠️ 注意：<code>-PredictionSource</code> 和 <code>-PredictionViewStyle</code> 需要 PSReadLine <strong>2.2+</strong> 版本才支持。</p>
</blockquote>
<h3 id="62-terminal-icons文件图标">6.2 Terminal-Icons（文件图标）</h3>
<p>在 <code>ls</code> / <code>dir</code> 命令时显示文件类型图标。</p>
<pre><code class="language-powershell">Install-Module -Name Terminal-Icons -Repository PSGallery -Force
</code></pre>
<p>在 Profile 中添加：</p>
<pre><code class="language-powershell">Import-Module -Name Terminal-Icons
</code></pre>
<p>效果对比：</p>
<pre><code># 未安装
README.md
src/
package.json

# 安装后
 README.md
 src/
 package.json
</code></pre>
<h3 id="63-z目录快速跳转">6.3 Z（目录快速跳转）</h3>
<p>记录访问历史，快速跳转到常用目录。</p>
<pre><code class="language-powershell">Install-Module z -Force
</code></pre>
<p>使用示例：</p>
<pre><code class="language-powershell">z Desktop    # 跳转到 Desktop
z pro        # 模糊匹配跳转到最近访问的含 pro 的目录
</code></pre>
<h3 id="64-posh-gitgit-状态显示">6.4 posh-git（Git 状态显示）</h3>
<p>在提示符中显示 Git 分支和状态信息。</p>
<pre><code class="language-powershell">Install-Module posh-git -Scope CurrentUser -Force
</code></pre>
<p>在 Profile 中添加：</p>
<pre><code class="language-powershell">Import-Module posh-git
</code></pre>

<h2 id="8-完整-profile-配置参考">8. 完整 Profile 配置参考</h2>
<p>运行 <code>notepad $PROFILE</code> 打开，将以下内容替换或追加进去：</p>
<pre><code class="language-powershell"># ============================================
# PowerShell Profile 配置
# ============================================

# Oh My Posh 主题（修改 atomic 为你喜欢的主题）
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\atomic.omp.json" | Invoke-Expression

# Terminal Icons（文件图标）
Import-Module -Name Terminal-Icons

# posh-git（Git 状态）
Import-Module posh-git

# Z 目录跳转
Import-Module z

# PSReadLine 配置
Set-PSReadLineOption -PredictionSource History
Set-PSReadLineOption -PredictionViewStyle ListView
Set-PSReadLineKeyHandler -Key Tab -Function MenuComplete
Set-PSReadLineKeyHandler -Key UpArrow -Function HistorySearchBackward
Set-PSReadLineKeyHandler -Key DownArrow -Function HistorySearchForward

# 常用别名
Set-Alias ll ls
Set-Alias vim nvim

# 自定义函数：快速进入常用目录
function desk { Set-Location "$env:USERPROFILE\Desktop" }
function docs { Set-Location "$env:USERPROFILE\Documents" }

# 显示欢迎信息
Write-Host "Welcome back, $env:USERNAME!" -ForegroundColor Cyan
</code></pre>
<p>保存后运行：</p>
<pre><code class="language-powershell">. $PROFILE
</code></pre>

<h2 id="9-常见问题">9. 常见问题</h2>
<h3 id="q1提示无法加载文件因为在此系统上禁止运行脚本">Q1：提示"无法加载文件，因为在此系统上禁止运行脚本"</h3>
<pre><code class="language-powershell">Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
</code></pre>
<h3 id="q2图标显示为乱码或方块">Q2：图标显示为乱码或方块</h3>
<p>原因是字体未正确设置。检查：</p>
<ol>
<li>是否安装了 Nerd Font</li>
<li>Windows Terminal 中是否选择了对应字体</li>
</ol>
<h3 id="q3oh-my-posh-提示符变慢">Q3：Oh My Posh 提示符变慢</h3>
<p>某些主题信息较多会影响速度，可以：</p>
<ol>
<li>换用更轻量的主题（如 <code>pure</code>、<code>minimal</code>）</li>
<li>在主题 JSON 中关闭不需要的 segment</li>
</ol>
<h3 id="q4predictionsource-参数报错">Q4：<code>PredictionSource</code> 参数报错</h3>
<p>说明 PSReadLine 版本过低，运行以下命令升级：</p>
<pre><code class="language-powershell">Install-Module PSReadLine -Force -SkipPublisherCheck
</code></pre>
<p>然后<strong>重启终端</strong>即可。</p>
<h3 id="q5如何找到-settingsjson-的位置">Q5：如何找到 settings.json 的位置</h3>
<pre><code class="language-powershell"># Windows Terminal settings.json 路径
"$env:LOCALAPPDATA\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json"
</code></pre>
<p>或在 Windows Terminal 设置页面左下角点击 <strong>打开 JSON 文件</strong>。</p>
